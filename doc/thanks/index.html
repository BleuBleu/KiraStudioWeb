GeneralUser GS

The app comes bundles with an OGG-compressed version of the SoundFont to keep the size of the app relatively small. The compressed version was made using MuseScore's amazing sftools.

Freeverb

Freeverb is public domain reverb implementation from the early 2000s. Developed by "Jezar at Dreampoint", it is a great, easy to use, reverb implementation.

I do not think it has an official website anymore, this repo hosts a copy of it.

Other Projects

I want to acknowledge some other projects that, while not providing code or libraries directly, were either inspirations or useful tools during development.

MeltySynth

MeltySynth was the original code I first looked at when trying to implement SoundFont support years ago. While the app does not use any of its code, I do want to acknowledge and draw attention to this great little project as it was a huge inspiration.

Polyphone

Polyphone is the de-facto tool for SoundFont editing. I used it as a benchmark to compared against during development. Not only is it a great tool, but the website as a ever growing collection of user-generated SoundFonts, which makes it a great resource for KiraStudio as well.

FamiStudio

While KiraStudio is closed source at the moment, the code is an evolution of my other open-source project, FamiStudio. If you are curious how a slightly more primitive version of the app was put together, go check it out.
